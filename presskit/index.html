Description

ImaginationOverflow is a small indie studio from Portugal that develops and publishes games and game development tools. ImaginationOverflow started publishing apps back in 2011 and transitioned to games in 2016 with the release of Stellar Interface. Since then it continued to develop and publish in-house games, releasing tools for the Unity Asset Store and doing contract work on other games.
